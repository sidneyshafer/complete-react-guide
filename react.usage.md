## React App Usage
>Includes all React apps - excludes Next.js apps

Creating a new React app:
```
npx create-react-app project-name
```

Install dependencies:
```
npm install
```

Run dev server:
```
npm start
```

Build for production:
```
npm run build
```

Check for dependency updates:
```
npm i -g npm-check-updates
ncu -u
npm install
```
