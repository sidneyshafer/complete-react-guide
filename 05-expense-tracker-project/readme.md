![image](https://github.com/sidneyshafer/complete-react-guide/assets/66838571/6bed6920-f58c-49b9-a954-98f86bd6a0c0)

## Usage
Install dependencies in terminal:
```
npm install
npm start
```
To build for production:
```
npm run build
```
